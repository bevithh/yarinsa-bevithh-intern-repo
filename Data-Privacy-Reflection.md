# Reflection

_Steps to handle data securely in daily tasks_
1.Use unique passwords with password manager
2.Enable two factors authentication
3.Lock computer when aways
4.Verify sender addresses before responding

_How to store, share, and dispose of sensitive information safely_
Stores sensitive files in encrypted, password protected, or in reputable cloud storage solutions.
Do not store sensitive data on personal devices or public cloud storage
Ensure you are sharing with the intended recipient and use multi-factor authentication

_Common mistakes and how to avoid_
Weak or simple password: do not use simple or identical password across multiple platforms, implement MFA
Seding sensitive data to wrong person: double-check recipients before sending, turn off auto-complete for external addresses
Poorly secured devices: use encryption for all devices including usb drives, enfore screen locking policies
Ignoring privacy settings and updates: regularly update softwares
